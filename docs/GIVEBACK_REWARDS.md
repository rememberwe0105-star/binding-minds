# 🎁 GiveBack Rewards — DearGiver 리워드 시스템 기획서

> **"Give more. Earn more. Give even more."**
> 기부를 통해 보상을 받고, 그 보상이 다시 기부로 이어지는 선순환 시스템

> **마지막 업데이트: 2026-05-18**

## 1. 핵심 컨셉 (v2 — 성취감 중심으로 전환)

### 핵심 방향 전환

| 항목 | v1 (기존) | v2 (확정) |
|------|-----------|-----------|
| **리워드 핵심** | Giving Card (실질 크레딧 $2~$25) | **감정적 성취감 + 이쁜 뱃지 수집** |
| **실질 보상** | 모든 중급 메달부터 크레딧 지급 | **극히 일부 최상위 tier에서만 (또는 Phase 3)** |
| **메시지 톤** | 비교 우월감 ("남들보다 당신이 낫다") | **임팩트 중심 ("당신의 기부가 만든 변화")** |
| **재원 부담** | 리워드 풀 $1,500/월 필요 | **거의 $0 (순수 디지털 배지)** |
| **구현 난이도** | 크레딧 결제 연동, 잔액 관리 필요 | **대시보드 UI + Firestore만으로 가능** |

### 핵심 원칙

> [!IMPORTANT]
> 1. **Cause-First** — 리워드는 보조 수단, 기부의 보람이 주인공
> 2. **임팩트 톤** — "남들보다 낫다"가 아닌 "당신의 기부가 만든 변화"
> 3. **공평한 접근** — 금액이 아닌 빈도/다양성 기반
> 4. **통계 진실성** — 모든 수치는 공식 출처 또는 플랫폼 내부 데이터만 사용
> 5. **실질 보상 최소** — 대부분 순수 디지털 배지, 크레딧은 극히 제한적

### 포지셔닝: "게임 보상"이 아닌 "기부 이력서"

뱃지 = 나의 기부 여정 기록
- "내가 어떤 분야에 관심을 가져왔는지" 시각화
- 자기 정체성 강화 효과 (심리학: Self-signaling theory)
- "나는 환경에 관심 있는 사람" → 환경 뱃지 축적 → 더 환경 기부

## 2. 수익 모델: 고정 $1 플랫폼 수수료 ✅ 확정

### 핵심 원칙

> **"우리는 기부금에 손대지 않습니다. 운영을 위해 딱 $1만 받습니다."**

- **부과 대상**: 자선단체 (기부자 추가 부담 $0)
- **금액**: 건당 고정 **$1 NZD**
- **최소 기부금**: **$20** ($1 = 5% 이하 보장)
- **도입 시기**: **처음부터** (무료 기간 없음, 투명한 신뢰 구축)

### 결제 흐름

```
기부자가 $50 기부 버튼 클릭
  → 기부자에게 $50만 청구 (추가 비용 $0) ✅
  → Stripe 수수료 ~$1.63 차감
  → DearGiver 플랫폼 수수료 $1.00 수취
  → 자선단체에 $47.37 전달
  → 세액공제 영수증: $47.37 기준 발행
```

### 수수료 비율 (기부금 대비)

- $20 기부 → $1 = 5% (최대)
- $50 기부 → $1 = 2%
- $100 기부 → $1 = 1%
- $200 기부 → $1 = 0.5% (최소)

### 왜 처음부터 $1인가?

> [!IMPORTANT]
> **무료→유료 전환은 거의 항상 실패한다.**
> - 단체가 "원래 무료였는데 이제 돈 내라고?" → 배신감
> - 한 번 무료로 인식되면 유료 전환 비용이 수수료 자체보다 큼
> - 처음부터 "딱 $1"은 투명성의 상징 → 오히려 신뢰 구축

### 자선단체 유치 메시지

> *"DearGiver는 기부 건당 딱 $1만 받습니다.*
> *$50 기부 기준 2%. GiveBack Rewards로 기부자 재방문율을 높여*
> *장기적으로 더 많은 기부를 유도합니다."*

### 경제성 시뮬레이션 (월 1,000건 기부)

- 예상 월 수익: **$1,000/월**
- Stripe 수수료, 서버 비용 등은 별도
- 리워드 풀 재원 거의 불필요 → 대부분 순수 운영비

### Stripe 수수료 부담: 기부자 선택형 ✅ 확정

Givealittle과 동일한 **체크박스 선택형** (NZ 사용자에게 익숙한 표준 방식)

```
☐ 결제 수수료를 부담하여 기부금 전액이 단체에 전달되도록 합니다
  - 체크 시: 기부자 $51.63 지불 → 단체 $49
  - 미체크 시 (기본값): 기부자 $50 지불 → 단체 $47.37
```

- **기본값**: 미체크 (기부자 추가 부담 $0)
- 업계 평균 60~70%가 체크 → 단체 수령액 자연스럽게 상승

### 미결정 사항

- [ ] 세액공제 영수증 금액 = 단체 수령액 기준 (확인 필요)

## 4. 뱃지 시스템 (v2)

### 🏅 메달 (Medals) — 영구 업적 배지

영구적으로 프로필에 표시되는 수집형 업적. **기부 습관과 다양성**을 보상합니다.

#### Beginner Tier (초보) — 순수 배지만
| 메달 | 조건 | 보상 |
|------|------|------|
| 🌱 Welcome Giver | **회원가입 완료** (Cold Start 해결) | 프로필 배지 |
| ✨ First Light | 첫 기부 완료 | 프로필 배지 |
| 🔍 Explorer | 2개 이상 카테고리에 기부 | 프로필 배지 |
| 🏠 Neighbour | 본인 지역 프로젝트에 기부 | 프로필 배지 |

#### Intermediate Tier (중급) — 순수 배지만
| 메달 | 조건 | 보상 |
|------|------|------|
| 💚 Steady Heart | 3개월 연속 기부 | 프로필 배지 |
| 🌉 Bridge Builder | 5개 다른 자선단체에 기부 | 프로필 배지 |
| 💯 Centurion | 누적 기부 $100 달성 | 프로필 배지 |
| 🎯 All-Rounder | 6개 카테고리 모두 기부 | 프로필 배지 |

#### Advanced Tier (상급) — 배지 + 미니 보상
| 메달 | 조건 | 보상 |
|------|------|------|
| 🥝 Kiwi Champion | 12개월 연속 기부 | 프로필 배지 + 연말 특별 기부 리포트 |
| ⭐ Thousand Star | 누적 기부 $1,000 달성 | 프로필 배지 + 프리미엄 프로필 프레임 |
| 👥 Community Pillar | 친구 5명 초대 (각각 기부 완료) | 프로필 배지 |
| 🌍 Impact Leader | 10개 이상 프로젝트에 기부 | 프로필 배지 |
| 🏆 Legendary Giver | 다른 모든 메달 수집 완료 | 프로필 배지 + 한정 프레임 |

> [!NOTE]
> **공평성 원칙**: 대부분의 메달은 금액이 아닌 **빈도와 다양성** 기반입니다.
> $5/월 기부자도 $500/월 기부자와 동일한 메달을 획득할 수 있습니다.
> 
> **Cold Start 해결**: 가입 즉시 "Welcome Giver 🌱" 배지 자동 발급
> + "첫 기부까지 딱 1번!" 다음 스텝 안내

### 🎄 Annual Giving Wrap (연말 기부 요약) — 🔥 핵심 기능
- Spotify Wrapped 스타일: "2026년, 당신의 기부 이야기"

### 🌿 기부자 등급 (Tier) 시스템
- 뱃지 수에 따른 프로필 시각 변화:
  - Seedling (0~2개) → Grower (3~5개) → Bloomer (6~9개) → Guardian (10+)
- 등급별 프로필 프레임/색상 변화
- **빈도/다양성 기반으로만** — 금액 기반 VIP처럼 보이면 안 됨

### 🤝 커뮤니티 챌린지
- 개인 경쟁이 아닌 **집단 목표**: "이번 달 500명이 환경 기부하면 전원 보너스 배지!"
- 달성 시 한정 배지 발급

## 9. 윤리적 가드레일 (Social Responsibility)

### ❌ 하지 않을 것

| 안티패턴 | 이유 |
|----------|------|
| 리워드를 메인 CTA로 사용 | 기부 동기가 보상이 되면 안 됨 |
| 금액 기반 랭킹 공개 | 기부자 간 불필요한 경쟁 유발 |
| 리워드 만료 임박 알림 폭탄 | 압박감 유발, 다크패턴 |
| 출처 없는 통계 사용 | NZ Fair Trading Act 위반 리스크 |
| 기부 빈도 압박 메시지 | 경제적 어려움 시 죄책감 유발 |

### ✅ 지킬 것

| 원칙 | 구현 |
|------|------|
| **Cause-First** | 리워드는 대시보드 서브탭, 메인은 항상 임팩트 스토리 |
| **공평한 접근** | 대부분의 보상은 빈도/다양성 기반, 금액 무관 |
| **투명성** | 공식 출처 통계만 사용, 출처 명시 |
| **커뮤니티 목표** | 개인 경쟁 아닌 집단 챌린지 |
| **누적 기반** | 기간 압박 없이 언제든 돌아와도 이어갈 수 있음 |

## 10. 소셜 공유 ✅ 확정

### 방향: "자랑"이 아닌 **"함께해요"** 톤

| 공유 가능 | 공유 불가 |
|-----------|------------|
| ✅ Annual Giving Wrap (연말 요약) | ❌ 개별 기부 금액 |
| ✅ 기부자 등급 달성 ("Guardian 등급!") | ❌ 누적 기부 총액 |
| ✅ 특정 메달 해제 ("Kiwi Champion!") | ❌ 기부 대상 단체 목록 |

- **톤**: "나도 기부하고 있어요, 함께해요 🌿" (참여 독려)
- **구현 시기**: Phase 2 (Annual Wrap과 함께)
- **금액 절대 비표시** — 활동 기반 메시지만 공유
